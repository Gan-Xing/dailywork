# Codex Prompt Guide

## 用途
记录与本项目相关的 AI/自动化使用约定，确保后续多人协作时可以快速复制环境和提示词。

## 使用原则
1. 开始任何任务前，先阅读 `docs/PROJECT_PLAN.md`，确认最新需求与步骤。
2. 若需求不明，先在“待确认输入”中列出问题，再向需求方收集信息。
3. 输出内容保持中文，除非项目计划中明确要求其它语言。
4. 不直接贴业务代码时，聚焦流程、接口、数据结构说明，避免冗长解释。

## 常用提示模板（可根据上下文调整）
```
基于 docs/PROJECT_PLAN.md 的最新约定，帮我完成【任务描述】，若信息不足请先列出需要的补充材料。
```

## 文档维护
- 每次引入新的协作规则或 AI 使用方式，追加到本文件。
- 若任务需要新的模板或上下文，增加对应提示块并注明适用场景。
